# 要件

無限ホワイトボードのWebアプリを作りたいです。

## 機能要件

* オブジェクトを任意の場所に配置できる。その後、オブジェクトを右クリックすると移動、変更、コピー、ペースト、複数選択、削除ができるポップアップメニューが出る
* オブジェクトには種類があり、図形（四角、太い矢印、細い線の矢印、円など）、文書、リンク、画像がある
* 文書オブジェクトはタイトルと本文をインプットとし、基本的にタイトルが表示されており、マウスオーバーするとその下にツールチップとして本文が表示される。
* リンクオブジェクトはURLをインプットとし、URL内のメインとなる画像が表示されており、マウスオーバーするとその下にツールチップとしてURLアクセス先のページが表示される。クリックするとそのページを別画面で開く
* 画像オブジェクトはタイトルとURLをインプットとし、URLから取得される画像が表示されており、マウスオーバーするとその下にタイトルが表示される。
* デザインは最新のマットな質感を希望 * 最終的にはスマートフォンアプリに展開したい（マウスオーバーは長押しで代替する）
* データはサーバまたはローカルのどちらかに保存できるようにして、最終的には同時編集可能としたいが、一旦は編集する際にロックをかける仕組みとして、1人しか更新できないようにする
* コンセプトとしてはエンタメ的なものを比較する際につかうプレゼンで利用することをイメージしており、ビジュアルに訴えられるようにしたい
* 案比較のためのVS表示やスコアリング表示も行える機能を拡充したい
